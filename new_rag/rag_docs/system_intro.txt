【系统简介】
边坡监测一体化系统用于实时监测城市/郊野边坡的位移、入侵和气象数据。
系统组成：
- 感知层：C8T6 单片机 + 毫米波雷达 + 摄像头 + 温湿度传感器 + 北斗/GPS 模块
- 通信层：ESP8266 AP 模式（热点名 ESP8266_AP，IP 192.168.4.1，端口 8000）传输距离数据；串口传输北斗定位
- 决策层：上位机运行 YOLOv8 入侵检测 + 基于大模型的 Agent 智能决策 + RAG 知识库问答
- 告警方式：微信（文件传输助手）和 QQ 邮件（SMTP）

系统特色：
- 纯本地部署，数据不公网传输
- 支持雨雾天气图像增强（convertScaleAbs 或 All-in-One Network）
- 支持短期滑动窗口 + 冷却时间，减少误报
- 支持远程控制：通过TCP指令控制LED和蜂鸣器（指令格式：CMD_LED_1_1_END... 等）。
- 上位机兼容操作系统：Windows 10/11，macOS（需安装Python依赖），Linux（Ubuntu 20.04+）。
